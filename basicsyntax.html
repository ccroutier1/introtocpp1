Syntax in C++

Basic Syntax of Hello World Program

Input-Output:Stream "#include iostream" - is an header file library that let us work with input and output objects, such as"cout". Header files add functionality to C++ Programs
Standard Library "using namespace std" - means that we can use names for objects and variables from the standard library.
Integer Main "int main()" - this is called a function any code inside its curly brackets {} will be executed
Console/Character Output "cout" - is an object used together with the insertion operator to output. Example: cout<<"Hello World";
Console/Character Input "cin" - it refers to "character" and "in" means "input". Thus, cin means "character input". The C++ cin object belongs to the istream class. cin is a predefined variable that reads data from the keyboard with the extraction operator >>. Example: cin>>var_name;
Return Console "return 0" - it means the value 0 (usually some typer of int) is returned from a function to the caller of the function.
