What is C++?

C++ is a high-level general-purpose programming language created by Danish Computer Scientist Bjarne Stroustrup as an extension of the C programming language, or "C with Classes".
The language has expanded significantly over time, and modern C++ now has object-oriented, generic, and functional features in addition to facilities for low-level memory manipulation.
C++ support Object-Oriented, Procedural and Generic Programming, C++ Is a Mid Level language as it emcapsulates both low level and high level features.
